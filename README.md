# What is this?
Basic linked list implemention using vanila Javascript

# Instalation
`npm i @truecodemedia/linkedlist`

Then ... 
``` 
import { LinkedList } from '@truecodemedia/linkedlist';

let list = new LinkedList();
```

# Available methods

+ `prepend`
+ `append`
+ `deleteFirst`
+ `deleteLast`
+ `addAtIndex`
+ `deleteAtIndex`
+ `deleteAtValue`
+ `contains`
+ `isEmpty`
+ `size`
+ `print`
